# OpenClash 自定义规则集

## 简介

这是一个专为 OpenClash 定制的自定义规则集，针对 AI 服务访问进行了优化，同时提供广告拦截和智能路由功能。

## 特性

- **AI 服务优化** - 专门针对 OpenAI、ClaudeAI 等 AI 平台的路由优化
- **广告拦截** - 内置丰富的广告域名拦截规则
- **智能分流** - 自动识别国内外服务，选择最优路由
- **自动测速** - 自动检测节点延迟并选择最优节点
- **肥猫云规则** - 自定义的代理规则集

## 文件结构

```
custom_openclash_rules/
├── add-ai-group.ini        # 主配置文件
├── fmy/                    # 自定义规则目录
│   ├── fmy.txt            # 完整规则定义
│   ├── fmy-DIRECT.list    # 直连规则列表
│   ├── fmy-Proxy.list     # 代理规则列表
│   ├── fmy-Pron.list      # 私密网站规则列表
│   └── fmy-REJECT.list    # 拦截规则列表
└── README.md
```

## 使用方法

### 在 OpenClash 中导入配置

1. 打开 OpenClash 配置页面
2. 进入「规则」或「覆写配置」选项
3. 将 `add-ai-group.ini` 的内容复制到「规则集自选」或对应区域
4. 确保启用以下配置：
   - `enable_rule_generator=true` (启用规则生成器)
   - `overwrite_original_rules=true` (覆盖原始规则)
5. 保存配置并应用

### 验证配置

1. 检查 OpenClash 规则页面是否显示了新的代理组和规则
2. 测试 AI 服务访问是否正常
3. 查看日志确认流量路由是否符合预期

## 规则优先级

规则按以下优先级匹配（从高到低）：

1. **拦截规则** - 广告和跟踪器域名
2. **直连规则** - 国内服务、本地网络
3. **代理规则** - AI 服务、国外网站
4. **兜底规则** - 未匹配的流量

## 规则列表说明

### 直连规则 (fmy-DIRECT.list)

包含以下类型的域名：
- 国内常用网站（淘宝、京东、百度等）
- Apple 服务（iCloud、App Store 等）
- 本地网络和局域网地址

### 代理规则 (fmy-Proxy.list)

包含以下类型的域名：
- AI 相关服务（Google、OpenAI 等）
- 社交媒体（Facebook、Twitter、YouTube 等）
- 开发者服务和工具

### 私密网站规则 (fmy-Pron.list)

包含以下类型的域名：
- 成人内容网站
- 私密和成人相关服务

### 拦截规则 (fmy-REJECT.list)

包含以下类型的域名：
- 各类广告域名
- 跟踪和统计服务
- 第三方分析工具
